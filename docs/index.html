New Eridu Companion

A security-first companion web app inspired by Zenless Zone Zero. It combines build/resource tracking with an AI assistant (RAG) and an “Attack Lab” to demonstrate defenses against prompt injection and abuse.

Focus: Secure full-stack patterns (rate limiting, audit logging) and LLM security techniques such as prompt injection and data exfiltration defenses.

Status: Building MVP v0 (notes + RAG-lite chat).

MVP v0
  • Notes page with basic retrieval
  • Chat page that answers using saved notes
  • Basic rate limiting + audit logging
